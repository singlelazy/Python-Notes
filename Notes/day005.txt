2.vim编辑器
	1.3种基本模式
		命令行模式：
			复制粘贴：
				dd:		删除光标所在那一行	
				ndd:	n为数字，删除光标所在的下n行
				yy:		复制光标所在的那一行
				nyy:	n为数字，复制光标所在的下n行
				p:		将已复制的数据在光标下一行粘贴
				P：		将已复制的数据在光标上一行粘贴
			撤销恢复：
				u:		撤销上一步操作
				ctrl+r:	恢复上一步操作
			光标移动：
				$:		跳到尾部
				0：		跳到头部
				ctrl+a：跳到头部(终端)
				ctrl+e；跳到尾部(终端)
				G：		跳到文件的最后一行
				nG:		n为数字，跳到文件的第n行
				gg:		移动到文件的第一行
				H：		移到屏幕最上方那行的第一个字符
				M：		移到屏幕中间那行的第一个字符
				L：		移到屏幕最下方那行的第一个字符

			
			
		编辑模式：输入a,i,o进入,输入Esc退回命令行模式
		扩展模式：输入:进入,输入Esc退回命令行模式
			w：		存盘
			q：		退出(不保存)
			wq!：	强制存盘退出
			set nu	显示行号
3.系统的初始化与服务
	1.启动顺序
		BIOS->MBR->Kernel->init
		MBR 512B
			446 引导信息
			64	分区信息
			2	标志位
		cat /etc/redhat-release 查看操作系统信息
		uname -r 查看内核版本
		uname -a 查看内核详细信息
		runlevel 查看启动级别
		init 0-6
			0	关机
			1	单用户模式，只有root用户能登陆，用于系统维护
			2	多用户模式，但是没有网络功能
			3	多用户模式，有网络功能，但是没有图形界面
			4	用户自定义模式，默认与3相同
			5	与3相同，但是有图形界面
			6	重新启动系统
	2.GRUB程序和grub.conf文件
		GRUB的定义：GRUB是Grand Unified Bootloader（多重操作系统启动管理器）
		grub-md5-crypt grub加密 
		bios 加密 bios里面security
4.操作系统进程管理
	内存中的buffer和cache
		buffer 缓存区
		cache  高速读取区
	1.进程:进程是程序的一次动态执行
		守护进程:守护进程是在后台运行并提供系统服务的一些进程
		父进程,子进程:当一个进程创建另一个进程时，第1个进程被称为新进程的父进程，而新进程被称为子进程
	2.top:能够实时显示系统中各个进程的资源占用状况,类似于Windows的任务管理器,系统默认每5秒刷新一下屏幕上的显示结果
		12:38:33		当前时间
		up 50days		系统运行时间
		1 user			当前登录用户数
		load average: 	0.06, 0.60, 0.48   系统负载。 三个数值分别为  1分钟、5分钟、15分钟前到现在的平均
		Tasks: 29 total	进程总数
		1 running		正在运行的进程数
		28 sleeping		睡眠的进程数
		0 stopped		停止的进程数
		0 zombie		僵尸进程数
	3.free:显示内存的使用状态
		-m  以M为单位
		-h	以G为单位
	4.ps:用来显示当前进程的状态
		aux				显示所有的与用户相关的完整信息
		grep 			过滤
			-v  		反向匹配
		pgrep+程序名	过滤程序的pid
		pstree  		以树状图显示程序
	5.kill:杀死进程
		-9+pid 			强制杀死
		pkill+程序名	全杀
	6.jobs:查看在后台工作的程序
		fg %工作号		调用到前台工作
	7.程序名+&			默认后台工作
